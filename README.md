## Setup Instructions
1. Clone the repository on your local device.
2. Open VSCode.
3. Open the terminal and navigate to a folder outside the connectly_folder.
4. Set up a virtual environment and download the dependencies. Refer to the [Activity 1 worksheet](https://docs.google.com/document/d/1mkqqK_UTTfJrloU0EZOXmlxEnx8xusfVztBB3Rqstmc/view?tab=t.0#heading=h.knscgki6s6cv).
5. Still using the terminal, navigate to the connectly_folder.
6. Run the server:
```
py manage.py runserver
```
